## Node Server

### Install the Server
Execute these commands to install the server:
```bash
cd ./server
npm install
```

### Run the Server
Execute these commands to run the server:
```bash
cd ./server
npm start
```

### Kill the Server
Execute this command to kill all node servers:
```bash
pkill node
```

### List Running Processes
```bash
ps -A
```
OR
```bash
ps aux
```
